# GenBridge Website

## Overview
GenBridge is a reverse mentorship platform that connects Gen Z experts with professionals seeking to enhance their digital skills. This website serves as the primary interface for users to explore mentorship opportunities, pricing plans, and market insights.

## Project Structure
The project is organized as follows:

```
genbridge-website
├── public
│   ├── index.html          # Main entry point for the website
│   ├── gen.html           # Additional content for mentorship and pricing
│   ├── css
│   │   └── main.css       # Styles for the website
│   └── js
│       └── main.js        # JavaScript for interactive elements
├── data
│   └── market-research.md  # Market research details and insights
├── .gitignore              # Files to be ignored by Git
├── package.json            # npm configuration and dependencies
└── README.md               # Project documentation
```

## Features
- **Mentorship Matching**: Users can find mentors based on specific topics of interest.
- **Pricing Plans**: Flexible subscription options for individuals and corporate clients.
- **Market Insights**: Detailed market research to support the business model and strategy.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd genbridge-website
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## Usage
- Open `public/index.html` in a web browser to view the website.
- Explore the mentorship options and pricing plans.
- Review the market research in `data/market-research.md` for insights into the mentorship industry.
